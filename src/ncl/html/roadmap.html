Version 2.0

Version 2.0 has been posted at Paul Lewis' software site for years. Bugs in this version are being fixed in the svn repository at https://svn.sourceforge.net/svnroot/ncl/branches/v2-stable

Version 2.1

A minor upgrade to version 2.1 is found at https://svn.sourceforge.net/svnroot/ncl/branches/v2.1. This version will change the original API as little as possible.

Programmers that do not have code that depends on the details of version 2.0 are encouraged to use 2.1

We will try to make the migration from 2.0 to 2.1 easy for code that already depends on NCL.

Version 3.0

The next major upgrade will primarily be a merging of functionality from previously forked versions of NCL. This is not ready for prime time yet.
