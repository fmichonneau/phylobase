Class Description

MyToken is derived from NxsToken in order to provide a way to report printable NEXUS comments (the square-bracket comments in NEXUS files that begin with an exclamation point and are intended to be printed in the outout).

    void   OutputComment(const NxsString &msg)
       
Overrides the base class version. When a NEXUS printable comment is encountered in a NEXUS file, the comment is passed to the virtual function OutputComment in its argument 'msg'. This version outputs 'msg' directly to both the standard output (cout) and the output file (out).
